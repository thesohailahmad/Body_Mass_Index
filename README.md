# BMI Calculator

This is a beginner-level Python project that calculates Body Mass Index (BMI)
using user input and conditional statements.

## Features
- Take age and gender as input
- Takes weight and height as input
- Calculates BMI
- Show All Results
- Displays health category

- Python language
